## Prerequisites

- [Node.js](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/) ou [npm](https://www.npmjs.com/)
- [Git](https://git-scm.com/)

## Installation

``` bash
git clone
cd fleurs-schrodinger
yarn install
```

## Launch development server

``` bash
yarn dev
```

## Build

``` bash
yarn build
```
